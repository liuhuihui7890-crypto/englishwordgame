# 英语学习射击游戏开发计划

这是一个寓教于乐的射击游戏，旨在帮助玩家学习英语单词。

## 项目计划

### 第一阶段：项目初始化与基础搭建
1.  **技术选型**：确定使用`HTML5/JavaScript` + `Phaser` 框架。
2.  **创建项目结构**：建立基本的文件夹和文件 (`index.html`, `js/`, `css/`, `assets/`)。
3.  **创建开发文档**：将开发计划和步骤写入 `README.md`。

### 第二阶段：核心玩法实现
1.  **搭建游戏场景**：使用Phaser创建一个黑色的游戏世界。
2.  **加载和显示资源**：在底部中心显示玩家的“炮台”。
3.  **生成问题和目标**：
    *   在屏幕底部显示问题单词（例如“apple”）。
    *   在游戏区域随机位置生成多个移动的目标（包含正确的“苹果”和错误的“香蕉”、“橘子”等）。
4.  **实现射击**：
    *   监听鼠标点击事件。
    *   点击时，从炮台发射一颗子弹。
5.  **碰撞检测与计分**：
    *   检测子弹是否击中目标。
    *   如果击中正确目标，则得分，目标消失。
    *   如果击中错误目标，或者子弹飞出屏幕，则子弹消失。
6.  **单词循环**：一个单词完成后，自动出下一个单词。

### 第三阶段：视觉与音效增强
1.  **设计资源**：寻找或制作更精美的炮台、子弹、目标背景、游戏背景图片。
2.  **增加动画**：
    *   子弹发射时的火焰/闪光效果。
    *   击中目标时的爆炸/消失动画（粒子效果）。
    *   目标出现时的动画效果。
3.  **增加音效**：
    *   背景音乐。
    *   射击音效。
    *   击中正确目标的音效。
    *   击中错误目标的音效。

### 第四阶段：内容与功能扩展
1.  **建立词库**：创建一个`JSON`文件来存储英汉词对，方便扩展和管理。
2.  **完善游戏流程**：
    *   创建开始菜单界面。
    *   创建游戏结束界面，并显示最终得分。
3.  **增加难度**：随着分数的提高，目标移动速度加快，或者出现得更频繁。
4.  **美化UI**：显示得分、生命值等。

### 第五阶段：构建与部署
1.  **代码优化与打包**。
2.  **部署**：将游戏部署到网络上，让其他人也可以玩。
